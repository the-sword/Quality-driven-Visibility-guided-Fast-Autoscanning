    CONSOLE APPLICATION : project-sample Project Overview
========================================================================

/////////////////////////////////////////////////////////////////////////////
/////////////////////////////////////////////////////////////////////////////

project-sample is a test application for Artec 3D Scanning SDK.

This sample loads two scans from an Artec Studio project, then aligns them, 
runs global registration and makes a model from the scans. Finally it saves 
the model along with the scans to another Artec Studio project.

Please note that the original scans must not be aligned.
